# 🏇 賽馬娘戀愛模擬器 — 大和赤驥篇

> 純文字 AVG 戀愛模擬器，免除原作育成的煩躁感，專注於戀愛劇情體驗。

## 📖 遊戲簡介

以大和赤驥為主角的戀愛模擬遊戲。你將扮演特雷森學園的訓練員，與赤驥從初遇到告白，經歷完整的五章戀愛故事。每個選擇都會影響好感度，最終決定你們的結局。

### 🎮 遊玩方式

直接在瀏覽器開啟 `index.html` 即可遊玩，不需要任何安裝或伺服器。

## 🧩 開發說明

### 新增章節
1. 在 `src/data/chapters/` 建立 `chapterX.js`
2. 定義 `CHAPTER_X_SCENES` 物件
3. 在 `src/data/index.js` 新增：`Object.assign(GAME_DATA.scenes, CHAPTER_X_SCENES);`
4. 在 `index.html` 新增 `<script>` 標籤

### 新增角色
在 `src/data/characters.js` 新增角色物件，包含 `id`、`name`、`color`、`description`。
